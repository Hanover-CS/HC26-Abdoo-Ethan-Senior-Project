# Setup & Prerequisites

## Required Tools
To follow this tutorial, you must have:
- A GitHub account
- GitHub Codespaces enabled
- Basic terminal knowledge (commands like `cd`, `ls`)
- Python 3.12+ (preinstalled in Codespaces)

## Starting Your Codespace
1. Open your repository on GitHub.
2. Click the green **Code** button.
3. Select **Open with Codespaces**.
4. Choose **New codespace** or reopen an existing one.

Your development environment will load in the browser.
