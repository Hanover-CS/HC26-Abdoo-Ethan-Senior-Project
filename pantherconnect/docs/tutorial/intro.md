# Introduction

## Target Audience
This tutorial is for beginners who have never used Django but can follow step-by-step instructions.

## Learning Objectives
By the end of this tutorial, you will be able to:
- Understand how Django projects and apps are structured
- Create a Django app inside a project
- Register apps in a Django project
- Define URL routes
- Create simple views that return responses
- Run a local development server
